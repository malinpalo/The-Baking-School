# The Baking School
The Baking School Website is a plattform for people who are curiose about how to bake, that never has baked, that might think that baking is difficult and boring. The Baking School offers classes in easy and fun baking. 
Users of the website will get all the information they need about the school, how to sign up and how to contact the school.

## Features
* Navigation
  * A navigation bar with navigation links (Home, About, Sign Up and Contact) is located up on the top right of the page.
  * The navigation links takes you to the prefferable section on the page.
  * The navigations font stands out from the background and the  color is in contrast with the background.
  * The user can easily find information on the website using the navigation.

* The HomePage Section
  * The homepage shows the name of the school together with a picture of baking.
  * The homepage explains that users can join a class to learn how to bake and that baking can be easy and fun.
  * The homepage text is in huiahgua font and is tigheter with the image in contrast to the background. 
  * The user gets a clear information of what the website is all about.

* The About Section
  * The About section gives the information about why people should bake, why they should choose The Baking School and how the school works.
  * The about section text continues in uhfuasighf font that contrasts with the background.
  * The user gets clear information about what the site is and how the school works.

 * The Sign Up Section
   * The Sign Up Section contains a of a form with fields for users to fill in, Name, Lastname, E-mail.
   * The form has a submitt button in the end. Users most fill in all fields to be abel to submitt.
   * The Sign up section gives users the ability to sign up for a class.

 * The Contact Section
   * The Contact Section provides users with a telefon number and a e-mail adress to enccourage users to take contact.
   * The user can easily contact The Baking School with questions or other requirements.
   

* The Footer Section
  * The Footer Section includes icons that links the user to The Baking School on social media such as Facebook and Instagram.
  * The Footer Section allows user to get connected with The Baking School at social media.
